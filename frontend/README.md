# Lexilearn AI - Frontend

This is the frontend of the Lexilearn AI platform, built with React and Vite.

For the full project documentation, including features, tech stack, and comprehensive setup instructions, please see the [Root README](../README.md).

## Quick Start (Frontend Only)

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
